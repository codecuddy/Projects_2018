Notes

  • Overview
    • Scenes: think level of the game (but could be another element: like main menu)
    • Game Objects: the environment, cameras, lights, and all other elements in the scene
      • Components: define the behavior, appearance or function of a GameObject
      • Prefabs: are preconfigured GameObjects, which you can store and reuse
      • Tags: help you identify GameObjects
      • Layers: make it easy to group individual GameObjects together in order to apply rules
    • Unity Community
    • Assets: Elements that make up your game (artwork, music, sound effects, animations, etc). Assest Store
  • Windows
    • Scene View
      • Hierarchy Window
        • Lists all objects in your game (characters, effects, music) in alphabetical order
        • Can group certain objects together creating hierarchical order and assigning Parent and Child categories. Groups within groups are descendants of the parent.
        • When you click on a GameObject in Hierarchy Window it becomes active in Scene View
    • Game View: how it will look in platform specific build
    • Inspector Window
      • View and edit properties and settings of GameObjects
